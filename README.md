# Web-Based Quiz Management System

## Overview
A full-stack quiz management system developed using Java, Spring Boot, MySQL, HTML, CSS, and JavaScript.

## Features
- User Registration and Login
- Quiz Attempt and Submission
- Score Calculation
- Result Tracking
- Category-wise Questions
- Admin Question Management

## Database Design

### Users Table
- id
- name
- email
- password

### Categories Table
- id
- category_name

### Questions Table
- id
- question_text
- option_a
- option_b
- option_c
- option_d
- correct_answer
- category_id

### Results Table
- id
- user_id
- score
